General questions

How long will Formula View be available for free?
Honestly, we don't know. We reserve the right to charge or require licenses for its use, but will give fair notice if we choose to do so.
Can I use Formula View offline?
Unfortunately not. Formula View, like all add-ins for Office 365, can be used only when connected to the web. Furthermore, the math view uses a web-based typesetting service provided by MathJax.org. Hence it is available only when it can reach the MathJax site.
Why doesn't Formula View work in Excel for the web with my browser?
Formula View does not work with Internet Explorer after Release 11.657, which fails to typeset the math view and fails to draw boxes in the tree view that are wide enough to contain symbols such as SQRT. Please follow Microsoft's recommendation to use Edge instead of Internet Explorer: it is faster and has better features. If you are having problems with another browser, please let us know.
Why doesn't Formula View recognize my formula?
Formula View does not recognize R1C1 notations for references. Neither does it recognize structured references, such as Inventory[InStock], to columns in tables. This may change in a future release.
Can Formula View misinterpret my formula?
Unfortunately, yes. Formula View treats identifiers such as ABCD2 and A1234567 as references to cells, and not as user-defined names, even though they exceed Excel's limits on the maximum numbers of rows and columns in a spreadsheet. Please let us know if it gets anything else wrong.

Questions about values in the tree view

Why won't Formula Forge show me values in the tree view?
Formula View will not compute values for the elements in your formula if it is on a protected worksheet. To ensure accuracy, Formula View temporarily writes most elements in your formula into its cell and gets their values from Excel. It cannot do this if the worksheet is protected. Unprotect the sheet if you want to see values.
But my worksheet isn't protected and I still can't see values in the tree view!
Formula View also refuses to compute values for a formula on a protected workbook. It hides its computation of values by displaying a new sheet named Formula Forge at work, and it cannot create that sheet if the workbook is protected. Unprotect the workbook if you want to see values.
Why does Formula View change the value of my formula?
As just noted, Formula Forge temporarily copies elements in your formula into its cell to get their values from Excel. This can change the value of your formula if it contains a volatile function, which generally produces a different result when it is re-evaluated. For example, RND deliberately produces different (and unpredictable) values. The functions NOW, TODAY, OFFSET, INDIRECT, and RNDBETWEEN are also volatile, as are INFO, CELL, and SUMIF for certain arguments. Excel recommends using volatile functions sparingly.
Why does Excel show inconsistent values in the tree view?
Formula Forge uses Excel repeatedly to compute values for all elements in a formula like RND()+1. In response, Excel produces one random value for RND() by itself and another for it in RND()+1. These are highly unlikely to differ by exactly 1. A future release of Formula View may evaluate volatile functions differently.
When else might Excel show incorrect values in the tree view?
Formula View may display incorrect values for expressions in array formulas or in functions like SUMPRODUCT, which treat their arguments as array formulas.
Can anything else go wrong if I ask Excel to show values in the tree view?
You should use Formula View with caution with the online version of Excel, which stores spreadsheets on OneDrive and automatically saves all changes. As a result, the temporary changes Formula View makes in your spreadsheets as it computes values will be visible on OneDrive. Likewise, you should use Formula View with caution with desktop versions of Excel when AutoSave is on.

Support

Can I get help with my problems?
To report a problem with Formula View, send email to support@formulaforge.com. Please tell us which version of Excel and which browser, if any, you are using. (You can find this information by choosing About in the drop-down menu in the Formula View pane.)
There seems to be a mistake in your documentation. Can you fix it?
We'll do our best if you send us email.
Can I ask for enhancements to Formula View?
By all means, yes. If what you request seems useful to others, and we can figure out how to do it, we will do our best to oblige.
