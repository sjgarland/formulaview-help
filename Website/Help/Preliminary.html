 Installation of Preliminary Release

Prior to Formula Forge being available in AppSource, users can ask to try a preliminary release by sending email to info@formulaforge.com. In response, they will receive a manifest, formulaforge.xml, to use as follows.

Using the preliminary release with Excel for the web

Each time Excel is run, open its Insert ribbon, choose Office Add-ins > MY ADDINS > Upload My Add-in > Browse. Then upload formulaforge.xml.

Using the preliminary release with Excel for the Macintosh

Put formulaforge.xml in ~/Library/Containers/com.microsoft.Excel/Data/Documents/wef.

Each time Excel is run, click the ▼ icon to the right of My Add-ins on Excel's Insert ribbon and select Formula Forge, which will be listed under Developer Add-ins.

Using the preliminary release with Excel for the Windows

Create an add-in catalog. In File Explorer, create a folder named AddInCatalog. Put formulaforge.xml in this catalog.

Share the add-in catalog. Right-click AddInCatalog and choose Properties. In the Properties dialog window, select the Sharing tab and click Share. Within the Network access dialog window, add yourself and click Share. When you see the confirmation Your folder is shared, click Copy to copy the network path for AddInCatalog to the clipboard. Then click Done to close this dialog window and OK to close the Properties dialog window.

Make Excel trust the add-in catalog. Click File on Excel's ribbon and select Options > Trust Center > Trust Center Settings. In the Trust Center dialog window, select Trusted Add-in Catalogs, paste the network path on the clipboard into the Catalog Url box, and delete the final right parenthesis and everything that comes before file:// in that box. Then click Add catalog and the Show in Menu check box for the newly trusted catalog. Finally, click OK twice to close the Trust Center and Excel Options dialog windows; then close Excel.

Now, each time Excel is run, open its Insert ribbon and choose My Add-ins. Select SHARED FOLDER at the top of the Office Add-ins dialog window, click Formula Forge, and then click Add.

Using the preliminary release with Excel for an iPad

Connect the iPad to a Macintosh with a USB cable. In iTunes on the Macintosh, click the small iPad icon in the menu bar to display the iPad's settings. Choose the File Sharing setting. Then choose Excel in the Add-ins column, click Add... in the Excel Documents column, and find and select formulaforge.xml.

Open Excel on the iPad. If it is already running, close and restart it. Choose Add-ins on the Insert ribbon and select Formula Forge from the Developer add-ins.
